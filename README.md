
# Nyka

## About the website

Nyka is an e-commerce website where all types of Skincare, Eyecare, Hairstyle and many more related products are available for men, women and kids.

## Technology
**Client** - Next.js, Redux, Chakra UI

**Server** - API, Vercel


## Features
- Signup and Login
- Whislist
- Cart 
- Payment
- Admin can add,
